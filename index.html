Technologies

Here are Languages I speak and Tools that I use everyday for Data Analysys!

Python

Python is an object-oriented programming language and contains various libraries and tools that can streamline the Data Analysis work. Such as Numpy, Pandas, Matplotlib, Scikit-Learn.

R

R is a software environment and statistical programming language built for statistical computing and data visualization. R's numerous abilities tend to fall into three broad categories: Manipulating data. Statistical analysis. Visualizing data.

Linux

From smartphones to cars, supercomputers and home appliances, home desktops to enterprise servers, the Linux operating system is everywhere. Linux is particularly strong in the field of open source statistical software.

SQL

SQL (Structured Query Language) is a powerful programming language helping data analysts interact with data stored in Relational databases.

Google BigQuery

BigQuery is a fully-managed, serverless data warehouse that enables scalable analysis over petabytes of data. Allowing you to quickly execute standard SQL queries and analyze millions of data rows in seconds.

Excel

Excel It is a very powerful data analysis tool and almost all big and small businesses use Excel in their day to day functioning. Empowers you to understand your data through high-level visual summaries, trends, and patterns.

Tableau

Tableau is a Business Intelligence tool for visually analyzing the data. Users can create and distribute an interactive and shareable dashboard, which depict the trends, variations, and density of the data in the form of graphs and charts.

Git

Git is a free and open source distributed version control system designed to handle everything from small to very large projects with speed and efficiency.
